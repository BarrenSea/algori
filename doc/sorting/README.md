# Sorting 排序

+ 插入排序
+ 选择排序

## 插入排序
插入排序将元素插入到已排序序列的正确位置 并扩展已排序序列的边界

[![Insertion_sort](./insertion_sort.svg)](./insertion_sort.svg)

### 复杂度
最优时间复杂度: O(n)
最坏时间复杂度: O(n ^ 2)

## 选择排序
选择排序按照指定的闭包每次找到

[![Selection_Sort](./selection_sort.svg)](./selection_sort.svg)

### 复杂度
选择排序的最优时间复杂度 平均时间复杂度和最坏时间复杂度均为 O(n^2)
